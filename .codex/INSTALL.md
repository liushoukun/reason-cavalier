# Install ReasonCavalier in Codex

Use this instruction in Codex:

```text
Fetch and follow instructions from https://raw.githubusercontent.com/liushoukun/reason-cavalier/refs/heads/main/.codex/INSTALL.md
```

## Manual Setup (Draft)

1. Clone this repository.
2. Copy or link:
   - `skills/`
   - `commands/`
   - `agents/`
   - `rules`/
   - `hooks/hooks.json`
3. Restart your Codex session and validate available modules.
