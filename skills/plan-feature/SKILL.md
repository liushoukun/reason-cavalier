# plan-feature

## Purpose

Help the agent convert a vague feature request into a clear implementation plan before coding.

## Trigger

Activate when:

- The user asks for a plan
- The task is ambiguous or broad
- There are architecture or trade-off decisions

## Behavior

1. Clarify goal and constraints.
2. Propose 2-3 implementation options with trade-offs.
3. Confirm the chosen option.
4. Produce a step-by-step execution checklist.

## Output Format

- Goal
- Constraints
- Options
- Recommended approach
- Execution checklist
