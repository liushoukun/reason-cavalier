# Claude Code Plugin Notes

ReasonCavalier exposes shared assets from this repository:

- `skills/` for reusable behavior modules
- `commands/` for command templates
- `hooks/` for automation hooks
- `rules`/ for rules

## Installation (Draft)

1. Register plugin source.
2. Install plugin by slug: `reason-cavalier`.
3. Start a new Claude Code session and verify skills are discoverable.

This file is intentionally minimal and will be expanded once distribution format is finalized.
