# init-project

Initialize a new plugin project using the ReasonCavalier baseline.

## Steps

1. Create required directories:
   - `.cursor-plugin/`
   - `.claude-plugin/`
   - `.codex/`
   - `skills/`
   - `commands/`
   - `agents/`
   - `hooks/`
2. Add baseline metadata files.
3. Add one sample skill and one sample command.
4. Verify repository structure.
